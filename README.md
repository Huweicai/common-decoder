# Common Decoder

An Alfred plugin that can almost decode everything by one command.

Current implemented decoders:

- UnixTimeStampDecoder
- UnixTimeStampMillDecoder
- UnixTimeStampNanoDecoder
- Base64Decoder
- Base64URLDecoder
- URLDecoder
- UnicodeDecoder
- DateTimeDecoder
- SpecialWordDecoder
- HexDecoder
- BinaryDecoder
- DecimalDecoder
- JWTDecoder
- CIDRDecoder
- StaticConfigDecoder
- UnitDecoder
